# ONU SDG Global Analyzer

## Project Description

This project analyzes Sustainable Development Indicators from the World Bank API.

The project integrates:

- High Performance Computing (Parallel Processing in Python)
- Cloud Services (Google Colab and Streamlit Community Cloud)
- Data Visualization (Plotly and Looker Studio)

## Technologies

- Python
- Pandas
- Requests
- Plotly
- Streamlit
- Google Colab
- Looker Studio
- GitHub

## Project Structure

- data/: Raw and processed datasets
- scripts/: Data processing scripts
- app/: Streamlit application
- dashboard/: Visualization resources
- docs/: Project documentation
